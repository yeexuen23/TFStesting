CHAPTER 29 Common Abstract Data Types

There are several abstract data types that are broadly useful in the JediPlus platform. They are:

containers

  • Set - a bitset
  • List - a doubly linked list
  • Queue -

number handling

  • Units - a representation of units and some methods to operate on them
  • Format
  • Math64

other

  • DateTime
