CHAPTER 5 Memory requirements

5-1 Overview

On February 11, 1998, the JediPlus software team convened to take a first pass look at the memory requirements for the JediPlus software system. We will record the results of the meeting in this file. This is not intended to be a comprehensive, extremely accurate accounting of the memory requirements. Instead, this document exists to account for the large portions of memory used in the design. The document is intended to be kept up to date as a quick reference to how the design is utilizing memory.
